# Модификация проекта

## Исходная версия

Первоначально бот использовал внешний API, который:

- работал нестабильно;
- поддерживал только прогноз на текущий день.

## Выполненные изменения

### Локальный словарь предсказаний

Создан собственный словарь предсказаний:

- 12 знаков зодиака;
- 8 дней;
- 96 записей.

### Inline-кнопки

Добавлены inline-кнопки для выбора даты:

- с 23 по 30 мая.

### Логирование пользователей

Реализовано логирование действий пользователей в файл:

- время;
- ID пользователя;
- действие.

### Автономность

Удалена зависимость от внешних API, благодаря чему бот стал полностью автономным.

## Результаты

После модификации проекта были получены следующие преимущества:

- повышение надёжности;
- расширение функционала;
- возможность анализа активности пользователей.
