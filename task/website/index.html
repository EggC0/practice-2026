🔮 Telegram-бот "Гороскоп на день"

Проект студентов: Тихонов Роман и Волынкин Артем
ВУЗ: Московский Политехнический Университет | Группа: №251-328

📌 О проекте

Бот позволяет получить ежедневный гороскоп для любого знака зодиака. Пользователь выбирает знак. Бот интегрирован с бесплатным API и написан на Python с использованием библиотеки pyTelegramBotAPI.

⚙️ Функциональность

✅ Команды

  • /start — приветствие
  • /horoscope — начать выбор
  • /help — справка

🎛️ Кнопки

Inline-клавиатуры для выбора знака. Никакого ручного ввода.

📡 API

Horoscope App API — возвращает текстовое предсказание и дату.

📊 Архитектура (диаграмма последовательности)

sequenceDiagram User->>Bot: /horoscope Bot->>User: Показать кнопки знаков User->>Bot: Выбор знака Bot->>User: Показать кнопки дней User->>Bot: Выбор дня Bot->>API: GET запрос API-->>Bot: JSON Bot->>User: Гороскоп

🖼️ Демонстрация работы

Скриншот 1: Старт

Команда /horoscope и появление кнопок знаков.

Скриншот 2: Выбор зз

Кнопки выбора знака задиака.

Скриншот 3: Результат

Полученный гороскоп с текстом и датой.
